# Nigeria House Prices Analysis

Exploratory analysis of Nigerian residential real estate, looking at what actually drives price and which affordability metrics hold up. The project pairs a Python notebook with a Power BI report and a written summary.

## Files

- `nigeria_house_prices_analysis.ipynb` — the analysis
- `nigeria-house-prices.pbix` — Power BI report
- `house-prices-summary-report.docx` — written findings

## What the data showed

Ikoyi is the priciest town in the Lagos data, averaging roughly ₦2.27 billion per property. Parking capacity doesn't scale with price the way you'd guess: eight-space properties sit as extreme outliers above ₦2.7B rather than on a smooth curve. Property title type moves median value in a real way. And price per bedroom turned out to be a steadier affordability signal than headline average price, which a few luxury listings distort badly.

## Stack

Python (pandas, Matplotlib, Seaborn), Jupyter, Power BI.
